What is JavaScript?

- Javascript named   Mocha > LiveScript > JavaScript

- weakly typed language

    let x = 'hello'
    x = 1

- object-oriented language

    let x = {
        name: 'John',
        age: 30
    }

- versatile language

    let x = 'hello'
    x = 1
    x = [1, 2, 3]

- JavaScript is a single threaded language (only one thing can be executed at a time.)


What is the difference between JavaScript and Java?

    - JavaScript is a client-side language  
    - Java is a server-side language

    - JavaScript is a compiled language
    - Java is an interpreted language

    - JavaScript is a high-level language
    - Java is a low-level language

    - JavaScript is a dynamic language
    - Java is a static language

    - JavaScript is a object-oriented language
    - Java is a functional language

    - JavaScript is a weakly typed language
    - Java is a strongly typed language

    - JavaScript is a versatile language
    - Java is a restrictive language

    - JavaScript is a single threaded language
    - Java is a multi-threaded language
    
- JavaScript is a dynamic language (variables can be reassigned)

    let x = 1
    x = 2

- It only has one call stack and therefore it is a synchronous language.
